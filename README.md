# Gowtham A V - Personal Portfolio Website

A modern, responsive personal portfolio website showcasing the professional profile of Gowtham A V, an ERP Executive at Jayashree Spun Bond, based in Madurai.

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Smooth Scrolling**: Enhanced navigation with smooth scroll effects
- **Interactive Elements**: Animated skill bars, scroll reveal effects, and hover animations
- **Contact Form**: Functional contact form for inquiries
- **Mobile-Friendly Navigation**: Hamburger menu for mobile devices

## Sections

1. **Hero Section**: Introduction with name, title, company, and location
2. **About Section**: Professional bio and highlights
3. **Experience Section**: Timeline of professional experience
4. **Skills Section**: Technical and soft skills with visual representations
5. **Contact Section**: Contact information and inquiry form
6. **Footer**: Quick links and social media connections

## Technologies Used

- HTML5
- CSS3 (with CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process or dependencies required!

## Customization

### Personal Information
Update the following in `index.html`:
- Name: Update throughout the file
- Company: Jayashree Spun Bond
- Location: Madurai
- Contact email: Update in contact section
- Social media links: Update in footer section

### Profile Photo
Replace the placeholder in the About section:
- Add your photo to the `images/` folder
- Update the image path in the About section

### Colors
Modify CSS variables in `css/style.css`:
- `--primary-color`: Main brand color (currently blue)
- `--primary-dark`: Darker shade for hover states
- Other color variables as needed

### Skills
Update skill percentages and names in the Skills section of `index.html`
